PDF Dossier Creator 2018

Combine folders, and sub-folders, of PDFs and retain file names and folder structure as bookmarks.

Do you need this?

You are using a directory to store and organized PDF files.

This directory contains multiple sub directories, or not.

Each directory/sub directory contains multiple PDF files, or just one.

How to use:

  1. Make sure all dependencies are installed.
  2. Double click to run CombineAndBookmark script file.
  3. Select source directory containing your PDFs.
  4. Wait for done message to appear.
  5. Admire your new combined PDF file in the parent directory of the source directory.

What it does:

  1. Prompt user to select source directory.
  2. Make copy of source directory in source directory parent folder.
  3. Open copy of source directory.
  4. For each PDF file in current directory and all sub directory...
    1. Delete existing bookmarks.
    2. Bookmark page 1 with file name.
  5. Open first sub directory, if any.
    1. If no sub directory, proceed to step 11.
  6. Repeat step 5 until no sub directory remain.
  7. Bookmark page 1 of each PDF in current directory with file name.
  8. Combine all PDF file in current directory into new PDF.
  9. Save the combined PDF to parent directory with current directory name.
    1. If current directory == copy of source directory, add "_combined" to output file name.
  10. Delete all PDF files in current directory.
  11. Delete current directory.
  12. Repeat steps 5 - 11 until copy of source directory is deleted.
  13. Final combined and bookmarked PDF is saved to source directory parent folder as [SourceDirectoryName]_Combined.

What it doesn’t do:

  • Read minds
    • The program is only as good as it's input.
    • PDF files in each directory must be numbered, [01, 02, 03, …] or [001, 002, 003,…], in the order they should be combined.
      • Otherwise, they will be combined in alphabetical order.
    • Sub directories should be numbered, [01, 02, 03, …] or [001, 002, 003,…], in the order they should be combined.
      • Otherwise, they will be combined in alphabetical order.
    • Name all PDF files and folder how they should appear in the table of contents.
  • Alter original files.
    • All changes are made to temporary copies of the files that are deleted as the program runs. Your origional files will be safe.
